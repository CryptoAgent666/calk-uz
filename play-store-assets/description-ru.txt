Calk.UZ — 78 бесплатных финансовых калькуляторов для жителей Узбекистана. Рассчитывайте налоги, зарплаты, кредиты, депозиты, коммунальные услуги, валюту и многое другое.

Расчёты основаны на открытых данных из официальных источников. Приложение использует публично доступную информацию для справочных расчётов.

ВОЗМОЖНОСТИ:
• 78 калькуляторов в 8 категориях
• Расчёты на основе актуальных открытых данных
• Интерфейс на русском и узбекском языках
• Без регистрации и сбора персональных данных
• Полностью бесплатно
• Работает офлайн

КАТЕГОРИИ:
📊 Налоги — НДФЛ, НДС, налог на прибыль, на имущество, земельный, IT Park
💰 Зарплата — net/gross, отпускные, больничные, компенсация при увольнении
🏦 Кредиты — аннуитет, дифференцированные платежи, рефинансирование
📈 Депозиты — сложные проценты, сравнение вкладов
💱 Валюта — конвертер по курсу ЦБ
🏠 Коммунальные — электричество, газ, вода, отопление
🏗️ Недвижимость — ипотека, аренда, ремонт
📐 Другое — проценты, маржа, точка безубыточности, возраст

Источники открытых данных, используемых для справочных расчётов:
• Центральный банк Узбекистана — cbu.uz (курсы валют)
• Налоговый комитет — soliq.uz (ставки налогов)
• База законодательства — lex.uz (нормативные акты)
• Министерство финансов — mf.uz (бюджетные данные)
• Госкомстат — stat.uz (статистика)
