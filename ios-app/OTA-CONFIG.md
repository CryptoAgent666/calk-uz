# OTA-конфиг для нативных калькуляторов

Нативные калькуляторы (Зарплата/НДФЛ, НДС, Растаможка) читают ставки из
**`https://calk.uz/app-config.json`** (файл лежит в репозитории сайта:
`public/app-config.json`). Это обновление **данных, не кода** — Apple такое
разрешает, и приложение полностью работает офлайн на вшитых значениях,
если файл недоступен.

## Как обновить ставку (OTA, без релиза в сторах)

1. Откройте `public/app-config.json` в репозитории calk.uz.
2. Поменяйте нужное число (например, `ndflStandard` с `0.12` на `0.13`).
3. Задеплойте сайт.
4. Установленные приложения подхватят новое значение **при следующем запуске**
   (и обновят уже открытый экран калькулятора через наблюдатель). Никакого
   обновления приложения и ревью Apple не требуется.

## Что можно менять (ключи `nums`)

| Ключ | Что это | По умолчанию |
|---|---|---|
| `ndflStandard` | НДФЛ, стандартная ставка | 0.12 |
| `ndflItpark` | НДФЛ для резидентов IT Park | 0.075 |
| `ndflNonresident` | НДФЛ для нерезидентов | 0.20 |
| `inps` | ИНПС (персональный пенсионный счёт) | 0.001 |
| `vatRate` | НДС | 0.12 |
| `customsDutyRate` | Таможенная пошлина (растаможка) | 0.15 |
| `customsVatRate` | НДС при растаможке | 0.12 |
| `brv` | БРВ, сум (меняется ежегодно!) | 412000 |

Расписания по объёму двигателя (массивы `[максимальный_объём_см³, значение]`):

| Ключ | Что это |
|---|---|
| `customsExciseBrackets` | Акциз, USD за см³ |
| `customsUtilBrackets` | Утильсбор, количество БРВ |

## Поведение / гарантии

- **Офлайн-safe:** если файла нет или сети нет — используются вшитые дефолты
  (те же числа). Калькуляторы 4.2-совместимы и работают без интернета.
- **Толерантность:** можно прислать частичный JSON — отсутствующие ключи
  берут значение по умолчанию. Лишние ключи игнорируются.
- **Кэш:** последний успешный конфиг кэшируется (UserDefaults), показывается
  мгновенно при следующем запуске, затем обновляется в фоне.
- **Источник:** `RemoteConfig.swift`; фетч в `AppDelegate.didFinishLaunching`.
  Тест-хук: env `CALK_CONFIG_URL` для подмены адреса конфига.

## ⚠️ Перед релизом

Файл `public/app-config.json` нужно **задеплоить на calk.uz**. Пока он не
задеплоен, приложение работает на вшитых дефолтах (это нормально). OTA
активируется, как только файл доступен по `https://calk.uz/app-config.json`.
